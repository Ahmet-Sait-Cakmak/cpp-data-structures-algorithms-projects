This repository contains my projects and exercises developed while taking a comprehensive **C++ Data Structures and Algorithms** course on Udemy.

The purpose of this repository is to reinforce theoretical concepts by implementing data structures and algorithms from scratch using C++.

## Topics Covered

### Linked Lists

* Singly Linked List
* Doubly Linked List
* Insert Operations
* Delete Operations
* Get/Set Node Methods

### Stacks

* Push
* Pop
* isEmpty
* Destructor Implementations

### Queues

* Enqueue
* Dequeue
* Queue Methods

### Binary Search Trees (BST)

* BST Node Class
* Insert Operations
* Destructor
* Memory Leak Exercises

### Graphs

* Graph Class
* Edge Insertion
* Vertex Removal
* Edge Removal

### Graph Traversal Algorithms

* Depth First Search (DFS)
* Breadth First Search (BFS)
* PreOrder Traversal
* InOrder Traversal
* PostOrder Traversal
